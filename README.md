<h2 align="center">
About This Project
</h2>
I created this project due to a need of more USB-C ports on my computer. However, I didn't just want any USB hub, I wanted one that represented me and my creativity. This USB Hub features 4 downstream ports, a power injection port for more power, and a fancy purple LED indicator light in the middle.

<h2 align="center">
Assembly
<br>
<img src="/assets/HexHubExplodedView.png">
</h2>


1. Place your PCB inside the bottom case
2. Screw the standoffs through the holes in the PCB and case and into the nuts below
3. Press the middle case into the bottom case
4. Place the acrylic on top of the middle case and screw it in with the bolts.
